# Wallet Dashboard — Think Forge Global LLP

A full-stack Wallet Dashboard built with React and Node.js + Express.

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Recharts, Axios
- **Backend:** Node.js, Express.js

## Features
- View total wallet balance
- Deposit & withdraw funds
- Add new bank accounts
- View transaction history
- Interactive wallet chart
- Fully responsive design


## Note
This project uses in-memory data storage to focus on clean API architecture 
and component design. MongoDB integration can be easily added for production use.

## Known Limitation
The app uses in-memory storage. On serverless platforms like Vercel, 
data may reset between requests. For persistent data, MongoDB integration 
is recommended for production deployment.
